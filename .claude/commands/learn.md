# /project:learn
Extract patterns and lessons from the current session for reuse.

After implementing a feature or fixing a bug, capture:
1. **Pattern identified:** what recurring FF pattern was used or discovered?
2. **When to apply:** trigger condition (e.g. "any new Kafka consumer in FF")
3. **Implementation:** the pattern in code with FF-specific context
4. **Pitfalls:** what went wrong first and why
5. **Save to:** `.ai/context/learned-patterns.md` (append)

This builds FraudFighter-specific institutional knowledge that persists across sessions.
